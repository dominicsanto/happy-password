# Happy Password
## A functional JS experiment

## Stack

- HTML
- CSS
- JavaScript
- RambdaJS

## Installation

`Clone the repo`

```
$ cd happy-password
$ npm install
```

## Usage

To start server:

```
$ http-server
```

and navigate to any of the listed URLs
